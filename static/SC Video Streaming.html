SC Video Streaming

This website demonstrates the 2 types of video streaming. In the "Stored" webpage, you can make Youtube search, select among the options, and watch the video you want. In the "Live" webpage, you can begin the live stream, pressing the Start option. Once you longer want to stream, simply select stop. Originally, a final project from my networks class where we only had to show a stored and live video, I added extra functionality to the application. Now, in order for visitors to use these services, they must either register a new account, or they can sign in with Google, using the Oauth 2.0 protocol.
